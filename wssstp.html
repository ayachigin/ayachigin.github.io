WebSocketSSTP

JavaScriptからSSPに通知を送れるようになるプラグインです。

var port = port || 55432;
var connection = new WebSocket('ws://' + HOST + port + '/notify');

connection.onopen = function () {
    connection.send('');
};

通知はディスプレイの右下に表示されます。 通知が表示されたときに通知音を鳴らすことも出来ます。
